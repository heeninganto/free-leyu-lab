# free-leyu-lab

一个用于归档和发布独立 HTML 页面的公共仓库。本仓库不针对任何具体网站或域名，仅作为静态 HTML 文件的存储与分发空间。

## 项目简介

本项目旨在提供一个干净、简洁的 HTML 页面存档环境。所有页面均为独立文件，不依赖外部数据库或服务端逻辑。适用于需要临时托管或分享静态页面的场景。

## 目录结构

```
free-leyu-lab/
├── pages/          # 存放已归档的 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件可独立访问，文件名建议使用有意义的英文或数字标识。
- `assets/`：若页面需要额外资源，请统一放置于此目录下，避免与页面文件混杂。

## 页面归档说明

- 所有页面均为静态 HTML，不包含后端交互功能。
- 页面内容应遵守相关法律法规，不得包含违法或侵权信息。
- 归档页面可能不定期清理或更新，请勿将其作为永久性存储依赖。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面，但请确保页面内容符合通用规范。
- 提交前请检查文件路径是否正确，资源引用是否完整。
- 仓库维护者保留对页面内容进行审核、修改或删除的权利。
